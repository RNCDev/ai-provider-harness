# @aph/react

Headless React hooks + optional unstyled primitives for AI Provider Harness.

- **Hooks:** `useProviders`, `useModels`, `useSelection`, `useSettings`, `useKey`, `useChat`
- **Primitives:** Radix-style unstyled `<ProviderSelect>`, `<ModelSelect>`, `<SettingsForm>`, `<KeyForm>`
- **Two runtimes, one tree:** `<HarnessProvider transport={...}>`

See [top-level README](../../README.md) for the full picture.
